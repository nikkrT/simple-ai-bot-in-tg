# Simple AI Telegram Bot

Простой и удобный Telegram-бот на Python, интегрированный с OpenAI API. Бот поддерживает диалоговый режим с возможностью настройки уровня логического вывода (reasoning effort), контроль доступа для пользователей и отслеживание дневных лимитов токенов.

## 🚀 Основные возможности

- **Интеграция с OpenAI:** Бот использует новые модели (с поддержкой `reasoning_effort`) для генерации ответов.
- **Уровни Reasoning:** Возможность переключаться между уровнями осмысления ответов нейросетью: `low`, `medium` и `high`.
- **Контроль доступа (Whitelist):** Пользоваться ботом могут только авторизованные пользователи.
- **Дневные лимиты токенов:** Встроенный подсчет использованных токенов за день (по UTC) для предотвращения перерасхода средств.
- **Обход лимитов Telegram на длину сообщения:** Если ответ ИИ слишком длинный, бот автоматически разобьет его на несколько сообщений с сохранением форматирования (включая блоки кода).
- **Раздельные промпты:** Поддержка обычных системных промптов и специальных ("risky") для определенных пользователей.

## 🛠️ Доступные команды

- `/start` — Приветственное сообщение.
- `/dialog` — Начать новый диалог с ботом.
- `/stop` — Очистить контекст текущей беседы.
- `/low` / `/medium` / `/high` — Установить уровень логического вывода (reasoning effort) для модели.
- `/tokens` — Проверить текущий расход токенов и оставшийся дневной лимит.

## ⚙️ Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/nikkrT/simple-ai-bot-in-tg.git
   cd simple-ai-bot-in-tg
   ```

2. **Установите зависимости:**
   Для работы бота требуется Python 3.x. Рекомендуется использовать виртуальное окружение.
   ```bash
   pip install -r requirements.txt
   ```

3. **Настройте конфигурацию:**
   Переименуйте файл `config_example.ini` в `config.ini` (или настройте по аналогии) и добавьте туда свои ключи и настройки:
   - Токен Telegram-бота (от @BotFather)
   - API-ключ OpenAI
   - ID пользователей, которым разрешен доступ к боту
   - Дневной лимит токенов (MAX_TOKENS_PER_DAY)
   - Название используемой модели (например, `o3-mini`, `gpt-4o` и т.д.)

4. **Запустите бота:**
   ```bash
   python main.py
   ```

## 📂 Структура проекта

- `main.py` — Главный файл запуска бота, обработка команд и сообщений.
- `init.py` — Инициализация конфигураций, ключей и списков пользователей из конфига.
- `message_cut.py` — Логика безопасного разделения длинных сообщений для Telegram с учетом Markdown.
- `_time.py` — Логика подсчета токенов и обработки дневных лимитов.
- `main_messages_and_promts.py` — Тексты команд, системные промпты и сообщения об ошибках.
